# Audio files

The backend serves `/audio` from `backend/public/audio/`. Scenario and phonics audio are in separate sections.

## Layout

- **`phrases/<lang>/`** – Scenario (phrase) TTS
  - One MP3 per card: `{cardId}.mp3` (e.g. `home-hello.mp3`, `market-water.mp3`, `clinic-i-feel-sick.mp3`).
  - Generated by `npm run generate-phrase-audio` (OpenAI TTS).
  - Served at `/audio/phrases/<lang>/{cardId}.mp3`.

- **`phonics/<lang>/`** – Phonics (letter/sound) audio
  - `phonics-*.mp3`, `placeholder.mp3`; optional `phonics-*.mp4` sources.
  - Fetched by `npm run download-phonics`; placeholders and custom recordings go here.
  - Served at `/audio/phonics/<lang>/{assetId}.mp3`.

- **`stories/`** – Advanced Story (Fable) dual-language audio
  - One pair per card: `{cardId}_en.mp3` (English), `{cardId}_rh.mp3` (Rohingya).
  - Example: `fable2_en.mp3`, `fable2_rh.mp3` for cardId `fable2`.
  - Served at `/audio/stories/{cardId}_en.mp3` and `/audio/stories/{cardId}_rh.mp3`.

## Scripts (run from backend)

- `npm run generate-phrase-audio` – generate phrase MP3s into `public/audio/phrases/en/`
- `npm run download-phonics` – download IPA phonics + placeholders into `public/audio/phonics/en/`
- `npm run convert-phonics-video` – convert .mov to .mp4 in `public/audio/phonics/en/`
- `npm run convert-phonics-audio` – extract .mp3 from phonics .mp4 in `public/audio/phonics/en/`
